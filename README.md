# s-modal

## Usage

### Install

```
yarn add s-modal

or

npm install s-modal
```

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build:wc
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
